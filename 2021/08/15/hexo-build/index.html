hexo建站过程实录

Raffi Zheng
2021-08-15
Web

平台及工具

  • Windows10系统电脑
  • git version 2.32.0.windows.2
  • node.js version 14.17.5
  • npm version 6.14.14

工具安装

git

  • 下载安装包后一路默认安装就好

node

  1. 安装时一路默认next,最好安装到非系统盘
  2. 在安装目录下新建两个文件夹”node_global”. “node_cache”,我是建在D盘下
  3. 打开cmd 分别执行以下两条指令,引号内的内容的按自己的目录更改
  • npm config set prefix “D:\nodejs\node_global”
  • npm config set cache “D:\nodejs\node_cache”
  1. 修改环境变量
  • 用户变量”path” -> 编辑最后一条npm -> 修改为”D:\nodejs\node_global”
  • 新建系统变量 -> 变量名”NODE_PATH” -> 值”D:\nodejs\node_modules”

test

  1. 打开cmd界面
  2. 分别输入”git -v”和”node -v”,观察是否反馈version号,若反馈则说明正常
